# Sample Upload Sets

Use one folder per tenant/business so your knowledge base stays clean and contextual.

## 1) Harbor Hotel (hospitality)
Create/register one tenant (for example: `harbor-hotel`) and upload all files from:
- `samples/harbor-hotel/`

Files:
- `hotel-operations.txt`
- `amenities-guide.md`
- `emergency-procedures.md`
- `housekeeping-standards.md`

## 2) City Cowork (workspace operations)
Create/register a different tenant (for example: `city-cowork`) and upload all files from:
- `samples/city-cowork/`

Files:
- `wifi-and-support.txt`
- `booking-rules.md`
- `membership-faq.txt`

## 3) Sunrise Dental (clinic admin)
Create/register a separate tenant (for example: `sunrise-dental`) and upload all files from:
- `samples/sunrise-dental/`

Files:
- `refund-policy.txt`
- `appointment-policy.md`
- `insurance-and-billing.txt`

## Upload recommendation
Do not mix files from different business folders into the same tenant.
